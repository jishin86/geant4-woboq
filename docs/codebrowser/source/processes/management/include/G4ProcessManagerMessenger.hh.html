31// G4ProcessManagerMessenger.hh
32//
33// Class Description:
34// This is a messenger class to interface to exchange information
35// between ProcessManager and UI.
36//-
37// /particle/process/ Process Manager control commands.
38// Commands :
39// dump * dump process manager information.
40// Verbose * Set Verbose Level for Process Manager and/or process
41// Activate * Activate process
42// inactivate * Inctivate process
43//
44// History:
45// 13 June 1997, H. Kurashige : The 1st version created.
46// 10 Nov. 1997 H. Kurashige : fixed bugs
47// 08 Jan. 1998, H. Kurashige : new UIcommand
48//
49//---------------------------------------------------------------
