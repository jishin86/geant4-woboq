32// G4Box
33//
34// Class description:
35//
36// A Box is a cuboid of given half lengths dx,dy,dz. The Box is
37// centred on the origin with sides parallel to the x/y/z axes.
38
39// History:
40// 30.06.95 P.Kent: Converted from source code developed end 94
41// 27.03.96 J.Allison: Added virtual functions DescribeYourselfTo() and
42// SendWireframeTo(G4VGraphicsModel&)
43// 22.07.96 J.Allison: Changed G4VGraphicsModel to G4VGraphicsScene
44// and SendPolyhedronTo() to CreatePolyhedron()
45// 27.03.98 J.Apostolakis: Inherit from G4CSGSolid (not G4VSolid)
46// 18.11.99 J.Apostolakis, V.Grichine: kUndefined was added to ESide
47// --------------------------------------------------------------------
