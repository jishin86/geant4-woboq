32// G4Ellipsoid
33//
34// Class description:
35//
36// A G4Ellipsoid is an ellipsoidal solid, optionally cut at a given z.
37//
38// Member Data:
39//
40// xSemiAxis semi-axis, x
41// ySemiAxis semi-axis, y
42// zSemiAxis semi-axis, z
43// zBottomCut lower cut plane level, z (solid lies above this plane)
44// zTopCut upper cut plane level, z (solid lies below this plane)
45
46// History:
47// -------
48// 10.11.1999 G.Horton-Smith (Caltech, USA) - First implementation
49// 10.02.2005 G.Guerrieri (INFN Genova, Italy) - Revision
50// --------------------------------------------------------------------
