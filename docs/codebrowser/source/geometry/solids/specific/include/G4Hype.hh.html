34// G4Hype
35//
36// Class description:
37//
38// This class implements a tube with hyperbolic profile.
39//
40// It describes an hyperbolic volume with curved sides parallel to
41// the z-axis. The solid has a specified half-length along the z axis,
42// about which it is centered, and a given minimum and maximum radius.
43// A minimum radius of 0 signifies a filled Hype (with hyperbolical
44// inner surface). To have a filled Hype the user must specify
45// inner radius = 0 AND inner stereo angle = 0.
46//
47// The inner and outer hyperbolical surfaces can have different
48// stereo angles. A stereo angle of 0 gives a cylindrical surface.
49
50// Authors:
51// Ernesto Lamanna (Ernesto.Lamanna@roma1.infn.it) &
52// Francesco Safai Tehrani (Francesco.SafaiTehrani@roma1.infn.it)
53// Rome, INFN & University of Rome "La Sapienza", 9 June 1998.
54//
