32// G4Paraboloid
33//
34// Class description:
35//
36// A G4Paraboloid represents a solid with parabolic profile with possible
37// cuts along the Z axis.
38//
39// Member Data:
40//
41// dz z half lenght
42// r1 radius at -dz
43// r2 radius at dz
44// r2 > r1
45//
46// Equation for the solid:
47// rho^2 <= k1 * z + k2;
48// -dz <= z <= dz
49// r1^2 = k1 * (-dz) + k2
50// r2^2 = k1 * ( dz) + k2
51
52// History:
53// -------
54// 10.07.2007 L.Lindroos (CERN) - First implementation
55//
56// --------------------------------------------------------------------
