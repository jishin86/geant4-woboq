29// class G4DisplacedSolid
30//
31// Class description:
32//
33// A displaced solid is a solid that has been shifted from its original
34// frame of reference to a new one. It is meant to be used only for
35// simplifying the implementation of "Boolean solids".
36
37// History:
38//
39// 28.10.98 V.Grichine: created
40// 22.11.00 V.Grichine: new set methods for matrix/vectors
41//
42// --------------------------------------------------------------------
