29// class G4ParameterisedNavigation
30//
31// Class description:
32//
33// Utility for navigation in volumes containing a single G4PVParameterised
34// volume for which voxels for the replicated volumes have been constructed.
35// [Voxels MUST be along one axis only: NOT refined]
36
37// History:
38// - Created. Paul Kent, Aug 96
39// --------------------------------------------------------------------
