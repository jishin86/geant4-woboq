29// class G4FieldTrack
30//
31// Class description:
32//
33// Data structure bringing together a magnetic track's state.
34// (position, momentum direction & modulus, energy, spin, ... )
35// Uses/abilities:
36// - does not maintain any relationship between its data (eg energy/momentum).
37// - for use in Runge-Kutta solver (in passing it the values right now).
38
39// History
40// - First version: Oct 14, 1996 John Apostolakis
41// - Modified: Oct 24, 1996 JA: Added dist_on_curve, deleted constructor
42// Nov 5, 1998 JA: Added energy, momentum, TOF, spin &
43// several constructor, access, set methods
44// May 10, 2006 JA: Added charge, "default" constructor
45// -------------------------------------------------------------------
46
