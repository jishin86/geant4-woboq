29// class G4QuadrupoleMagField
30//
31// Class description:
32//
33// Class for creation of quadrupole magnetic field
34// fGradient - is the gradient value for quadrupole magnetic lense.
35// Then the magnetic field components are:
36// Bx = B[0] = fGradient*X ,
37// By = B[1] = fGradient*Y ,
38// Bz = B[2] = 0 .
39// Here X,Y,Z are the coordinates of a space point of interest.
40
41// History:
42// 3.2.97 - V.Grichine, created.
43// 11.5.12 - B. Riese allow displaced origin and rotation (Bj√∂rn Riese)
44// -------------------------------------------------------------------
45
