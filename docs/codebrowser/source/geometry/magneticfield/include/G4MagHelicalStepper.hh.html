31// class G4MagHelicalStepper
32//
33// Class description:
34//
35// Abstract base class for integrator of particle's equation of motion,
36// used in tracking in space dependent magnetic field
37
38// It is used for a set of steppers which use the helix as a sort of
39// 'first order' solution.
40// - Most obtain an error by breaking up the step in two
41// - G4ExactHelicalStepper does not provide an error estimate
42
43// History:
44// - 05.11.98 J.Apostolakis Creation of new ABC
45// --------------------------------------------------------------------
46
