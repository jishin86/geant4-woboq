28////////////////////////////////////////////////////////////////////////
29// Class G4LogicalSurface
30////////////////////////////////////////////////////////////////////////
31//
32// Class description:
33//
34// An abstraction of a geometrical surface, it is an abstract
35// base class for different implementations of surfaces.
36// Its primary function is to hold pointers to objects that describe the
37// surface's physical properties. For example it holds a pointer to a
38// surface's optical properties, and because of this it is used in processes
39// like G4OpBoundaryProcess.
40//
41// Methods:
42// G4SurfaceProperty* GetSurfaceProperty() const
43// void SetSurfaceProperty(G4SurfaceProperty*)
44//
45// G4String GetName() const
46// void SetName(const G4String&)
47//
48// G4TransitionRadiationSurface* GetTransitionRadiationSurface() const
49// void SetTransitionRadiationSurface(G4TransitionRadiationSurface*)
50//
51// Data members:
52// G4String theName
53// G4SurfaceProperty* theSurfaceProperty
54// G4TransitionRadiationSurface* theTransRadSurface
55
56// Created: 1997, June, 4th to 17th
57// Author: John Apostolakis, (with help of Peter Gumplinger)
58// mail: japost@mail.cern.ch
59//
60// ------------------------------------------------------------------------
